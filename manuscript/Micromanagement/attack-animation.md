## 3.1 Анимация автоатаки

Каждый чемпион в League of Legends имеет уникальные **комбо** (combo). Так называется последовательность действий, которая увеличивает их эффект. Например, атакующее комбо увеличивает урон чемпиона в секунду (DPS).

Чтобы исполнить любое комбо, необходимо понять и освоить работу с анимацией чемпиона. Начнём с самой простой анимации — базовой атаки чемпиона.

I> Когда игрок даёт команду на любое действие, чемпион его исполняет. В это время он совершает какие-то движения. Они называются **анимацией действия**. Обычно анимация уникальна для каждого действия.

### 3.1.1 Скорость автоатаки

[**Базовая атака**](https://leagueoflegends.fandom.com/wiki/Basic_attack) или автоатака (basic attack, auto attack или AA) — это стандартный способ нанести противнику урон. Он доступен каждому чемпиону. Чтобы выполнить автоатаку, достаточно нажать правой кнопкой мыши по противнику.

Анимация автоатаки зависит от характеристики чемпиона, которая называется **общая скорость атаки** (total attack speed или total AS). Она складывается из следующих компонентов:

1. **Базовая скорость атаки** (base attack speed или base AS) чемпиона.

2. **Дополнительная скорость атаки** (bonus AS) за уровень чемпиона.

3. Bonus AS за предметы.

4. Bonus AS за руны.

5. Bonus AS за умения.

6. Bonus AS за усиления.

7. Штраф скорости атаки за ослабления.

Рассчитаем на примере общую скорость атаки (total AS). Возьмём чемпиона Ashe с base AS равной 0.66. На первом уровне без предметов, умений, рун на скорость атаки и усилений total AS чемпиона будет равна base AS. Это означает, что total AS равна 0.66.

Допустим, что Ashe покупает предмет [Berserker's Greaves](https://leagueoflegends.fandom.com/wiki/Berserker's_Greaves). Он даёт bonus AS 35%. Тогда total AS рассчитывается по следующей формуле:
{line-numbers: false, format: text}
```
total_AS = base_AS * (1 + bonus_AS / 100)
```

Подставим числа и получим:
{line-numbers: false, format: text}
```
total_AS = 0.66 * (1 + 35 / 100) = 0.89
```

I> У каждого чемпиона есть характеристика **коэффициент скорости атаки** (AS ratio). Это множитель для bonus AS. AS ratio у большинства чемпионов равен 1. Им предмет Berserker's Greaves даёт 35% AS. У некоторых чемпионов AS ratio меньше единицы (например, у Senna и Twisted Fate). Для них бонус от Berserker's Greaves окажется меньше 35%.

Мы рассчитали общую скорость атаки (total AS). Она определяет, сколько времени чемпион будет проигрывать анимацию одной автоатаки. Этот параметр называется **время перезарядки автоатаки** (attack cooldown). Он рассчитывается так:
{line-numbers: false, format: text}
```
attack_cooldown = 1 / total_AS
```

Подставим числа для Ashe с Berserker's Greaves и получим:
{line-numbers: false, format: text}
```
attack_cooldown = 1 / 0.89 = 1.12
```

Это означает, что между двумя автоатаками чемпиона должно пройти минимум 1.12 секунды. Другими словами, чемпион не выполнит вторую автоатаку, пока не пройдёт время перезарядки.

### 3.1.2 Фазы анимации автоатаки

Мы научились рассчитывать время перезарядки автоатаки. Пока оно идёт, чемпион проигрывает полный цикл анимации атаки. Цикл состоит из трёх фаз:

1. **Windup** — замах.

2. **Firing** — удар или выстрел.

3. **Recovery** — возврат в исходное положение.

Рассмотрим эти фазы на примере Ashe первого уровня без предметов. Её время перезарядки автоатаки равно 1.52 секунды:
{line-numbers: false, format: text}
```
attack_cooldown = 1 / 0.66 = 1.52
```

Иллюстрация 3-1 демонстрирует временную диаграмму одной автоатаки Ashe.

{caption: "Иллюстрация 3-1. Временная диаграмма автоатаки Ashe", width: "100%"}
![Временная диаграмма автоатаки](images/Micromanagement/ashe-attack-animation.png)

На иллюстрации изображена ось времени s. На ней отложены три отрезка. Каждый соответствует одной из трёх фаз анимации. Над отрезками приведены скриншоты из игры. Они демонстрируют кадры из соответствующей анимации.

В точке A игрок нажимает правой кнопкой мыши на цель. С этого момента чемпион проигрывает фазу анимации windup: натягивает тетиву лука. Она длится 21.93% от всего времени перезарядки автоатаки. Это равно 0.33 секунды, согласно следующему расчёту:
{line-numbers: false, format: text}
```
windup = 1.52 * 21.93 / 100 = 0.33
```

В точке B анимация windup заканчивается. После этого чемпион проигрывает фазу firing: выпускает стрелу в цель. Она длится всего 3.37% от времени перезарядки автоатаки. Это равно 0.05 секунд:
{line-numbers: false, format: text}
```
firing = 1.52 * 3.37 / 100 = 0.05
```

В точке C чемпион начинает фазу recovery: достаёт из колчана следующую стрелу. Эта фаза длится 74.22% времени перезарядки автоатаки. Она составляет 1.13 секунды:
{line-numbers: false, format: text}
```
windup = 1.52 * 74.22 / 100 = 1.13
```

В точке D чемпион заканчивает полный цикл анимации. Прошло 1.52 секунды с момента, когда игрок дал команду в точке A. Если чемпион не получит новой команды, то в точке D он начнёт следующий цикл анимации атаки по той же цели.

В любой момент на отрезка A-D игрок может дать чемпиону команду на движение. Её результат зависит от фазы анимации, которую сейчас проигрывает чемпион. Возможны [три случая](https://leagueoflegends.fandom.com/wiki/Stutter-stepping):

1. Отрезок A-B — команда попала на фазу windup:

   * Чемпион отменит автоатаку.

   * Цель атаки не получит урон.

   * Таймер перезарядки автоатаки сбросится в ноль.

   * Игрок может запустить следующую автоатаку в любой момент. Её анимация начнётся с фазы windup (точка A).

2. Отрезок B-C — команда попала на фазу firing:

   * Команда на движение попадёт в очередь.

   * Чемпион выполнит автоатаку.

   * Цель атаки получит урон, если атака в ближнем бою. В случае дальнего боя цель получит урон, когда в неё попадёт снаряд.

   * Таймер перезарядки автоатаки продолжит идти.

   * После окончания анимации firing (точка C) команда на движение извлекается из очереди. Чемпион её выполнит, вместо анимации recovery. Он идёт в указанную точку.

   * Игрок не может запустить следующую автоатаку. Сначала должен закончиться таймер перезарядки автоатаки в точке D.

3. Отрезок C-D — команда попала на фазу recovery:

   * Чемпион уже выполнил автоатаку.

   * Цель атаки уже получила урон.

   * Таймер перезарядки автоатаки продолжает идти.

   * Чемпион прекращает анимацию recovery и идёт в указанную точку.

   * Игрок не может запустить следующую автоатаку. Сначала должен закончиться таймер перезарядки автоатаки в точке D.

Из правил сочетания двух команд можно сделать следующие выводы:

1. Отмена фазы windup — это почти всегда ошибка. Она уменьшает урон чемпиона в секунду (DPS). Единственный случай, когда отмена оправдана, — игрок обнаружил опасность и вынужден немедленно отступать.

2. Фаза recovery всегда занимает большую часть времени перезарядки автоатаки. Позволять чемпиону её проигрывать — значит впустую тратить время. Вместо этого следует двигаться или использовать умение.

3. Самый эффективный способ отменить фазу recovery — дать команду во время firing. Тогда сразу после firing чемпион исполнит новую команду.

### 3.1.3 Атака и движение

Техника отмены анимации recovery командой движения называется **attack move** (атака и движение). Её хорошо объясняет [видео на канале Skill Capped](https://www.youtube.com/watch?app=desktop&v=-oyxOgtT33U).

Допустим, что игрок отменяет анимацию атаки командной движения. Рассмотрим, как в этом случае поменяет временная диаграмма анимации чемпиона. Её демонстрирует иллюстрация 3-2.

{caption: "Иллюстрация 3-2. Временная диаграмма attack move", width: "100%"}
![Временная диаграмма attack move](images/Micromanagement/ashe-attack-move.png)

Отрезки A-B и B-C остались такими же, как на иллюстрации 3-1. Мы знаем, что чемпион должен проиграть всю анимацию windup и firing. Только после этого он выполнит автоатаку и нанесёт урон противнику.

На иллюстрации 3-2 игрок даёт команду на движение на отрезке B-C диаграммы. Поэтому чемпион двигается сразу после завершения анимации firing в точке C. В нашем примере время движения составляет 1.13 секунды и соответствует отрезку C-D.

В точке D заканчивается таймер перезарядки автоатаки. Это означает, что теперь игрок может дать следующую команду на атаку. В этом случае чемпион начнёт новый цикл анимации атаки с фазы windup.

Какие преимущества даёт игроку техника attack move? Её применяют в двух случаях:

1. Для преследования противника и нанесения ему урона.

2. Для отступления и нанесения противнику урона.

В первом случае отступающий противник получает больше урона. Сравним, к чему приведёт одна команда автоатаки по отступающему противнику и выполнение техники attack move.

Если дать чемпиону **одну команду** на автоатаку, он будет стоять на месте и проигрывать все три фазы анимации атаки. Ashe потратит на это 1.51 секунд. Всё это время противник будет разрывать с ней дистанцию.

[**Дальность атаки**](https://leagueoflegends.fandom.com/ru/wiki/Дальность_атаки) (attack range) Ashe составляет 600. Поэтому когда она закончит анимацию фазы recovery, возможны два случая:

1. Противник удалился от чемпиона на дистанцию больше 600. В этом случае Ashe самостоятельно начнёт двигаться за противником.

2. Противник всё ещё находится на дистанции менее 600. В этом случае Ashe начнёт следующий полный цикл анимации атаки.

Допустим, что противник оказался вне дальности атаки Ashe. Пусть его [**скорость передвижения**](https://leagueoflegends.fandom.com/ru/wiki/Скорость_передвижения) (movement speed или MS) не меньше чем у Ashe. В этом случае она не сможет догнать цель и сделать ещё один выстрел. Противник уже отступил на безопасную для него дистанцию.

Если применить **технику attack move**, чемпион будет стоять на месте только во время фаз windup и recovery анимации атаки. Ashe потратит на это 0.38 секунды. Именно столько времени противник будет разрывать с ней дистанцию. Затем игрок даёт команду на движение в сторону противника. Тогда в течении следующих 1.13 секунды Ashe будет двигаться за целью. После этого её таймер перезарядки автоатаки закончится. Игрок даёт команду на атаку и чемпиона делает следующий выстрел.

В рассмотренной ситуации каждый выстрел обходится Ashe в 0.38 секунды. Это то время, которое она стоит на месте, а противник удаляется от неё. Всё остальное время между атаками Ashe двигается за целью.

Найдём соотношение времени движения противника в первом и втором случае:
{line-numbers: false, format: text}
```
1.51 / 0.38 ~ 3.97
```

Это означает, что с техникой attack move противнику понадобится почти в 4 раза больше времени, чтобы отступить на безопасную дистанцию. Если у него низкий уровень здоровья, то он не успеет этого сделать.

Если игрок применяет технику attack move для отступления, он уменьшает время, в течении которого его чемпион получает урон от противника. Идея этой техники точно такая же, как у attack move для преследования.

Сначала игрок даёт чемпиону команду на атаку цели. Ashe проигрывает анимации windup и firing в течении 0.38 секунды. После этого игрок даёт команду движения от противника. Ashe будет двигаться от него в течении 1.13 секунды. После этого её таймер перезарядки автоатаки закончится. Игрок  даёт команду на атаку и чемпион делает следующий выстрел.

К чему приведёт техника attack move при отступлении? Допустим, что противник — это чемпион ближнего боя с дальностью атаки 175. У Ashe дальность атаки гораздо больше и равна 600. После того как Ashe удалится от противника на 175, он прервёт свою атаку и будет вынужден двигаться за ней. Пусть скорость передвижения Ashe больше чем у цели. Тогда она будет успевать атаковать и отступать до того, как противник к ней приблизится на дистанцию 175. Таким образом он ни разу не сможет её атаковать. Эта техника называется **кайтинг** (kiting).

Если противник тоже использует технику attack move при отступлении или преследовании, он получит те же преимущества, что и игрок. В этом случае никто не получит выгоды. Но если игрок не станет применять эту технику, преимущество окажется у противника.

Рассмотрим самый эффективный способ исполнения техники attack move. Для этого надо изменить настройки игры и горячих клавиш.

Первая настройка находится во вкладке "HOTKEYS" и называется "Player Attack Move Click". Её демонстрирует иллюстрация 3-3

{caption: "Иллюстрация 3-3. Настройка 'Player Attack Move Click'", height: "50%"}
![Настройка Player Attack Move Click](images/Micromanagement/options-player-attack-move-click.png)

Это горячая клавиша для команды на автоатаку. Назначьте для неё клавишу A. Тогда каждый раз, когда вы нажимаете A, чемпион будет атаковать ближайшую к нему цель.

Вторая настройка находится во вкладке "GAME" и называется "Attack move on cursor". Её включение демонстрирует иллюстрация 3-4

{caption: "Иллюстрация 3-4. Настройка 'Attack move on cursor'", width: "100%"}
![Настройка Attack move on cursor](images/Micromanagement/options-attack-move-on-cursor.png)

Включите эту настройку. Теперь при нажатии клавиши A чемпион будет атаковать цель, ближайшую к курсору мыши. Без этой настройки вы не сможете выбрать правильную цель в командном сражении.

Теперь рассмотрим, как выполнять технику attack move с новыми настройками. Для этого повторяйте следующие шаги:

1. Приблизьте курсор мыши к нужной цели.

2. Нажмите горячую клавишу A. Чемпион начнёт проигрывать анимацию windup.

3. Когда чемпион перейдёт в анимацию firing или recovery, нажмите правую кнопку мыши. Чемпион начнёт двигаться в указанную точку.

4. Считайте в уме таймер перезарядки автоатаки. Когда он закончится, нажмите клавишу A. Чемпион снова начнёт атаку.

5. Продолжайте выполнять шаги 3 и 4.

При выполнении техники attack move важно не совершать следующие ошибки:

1. Давать команду на движение в фазу анимации windup. Тогда вы отмените автоатаку чемпиона и он не нанесёт урон противнику.

2. Позволять чемпиону двигаться дольше, чем идёт его таймер перезарядки автоатаки. В этом случае вы не реализуете его скорость атаки и уменьшаете DPS.

Тренируйте технику attack move в режиме игры "Practice tool". Научитесь интуитивно чувствовать ритм атаки своего чемпиона на разных уровнях и с разными предметами. Тогда вы сможете начинать новую автоатаку точно в момент окончания таймера её перезарядки.

Обратите внимание, что скорость атаки определяет дальность передвижения чемпиона между выстрелами или ударами. При низкой скорости атаки, чемпион может двигаться более длинными отрезками. При высокой скорости атаки эти отрезки укорачиваются.

### 3.1.4 Атака и применение умения

Фазу recovery анимации автоатаки можно отменить не только командой движения. Вместо движения можно использовать одно из умений чемпиона. В этом случае временная диаграмма будет выглядеть как на иллюстрации 3-5.

{caption: "Иллюстрация 3-5. Временная диаграмма отмены анимации атаки умением", width: "100%"}
![Временная диаграмма отмены анимации атаки умением](images/Micromanagement/ashe-attack-ability.png)

Отрезки A-B и B-C остались такими же, как на иллюстрациях 3-1 и 3-2. Это означает, что чемпион проигрывает фазы анимации windup и firing как обычно. После этого Ashe выпускает стрелу по противнику.

На иллюстрации 3-5 игрок даёт команду Ashe применить умение W на отрезке B-C диаграммы. Она начинает анимацию умения сразу после завершения анимации AA firing в точке C. Таким образом игрок отменяет анимацию recovery автоатаки, которая длилась бы 1.13 секунды.

Технику отмены анимации атаки умением применяют во время сражения. В зависимости от ситуации игрок должен выбрать, каким действием ему отменять анимацию: движением или умением. Умение выгодно применять, когда чемпион занял подходящую позицию и ему больше не нужно двигаться. Эта техника значительно увеличивает DPS чемпиона по неподвижной цели. Если цель двигается или все атакующие умения на перезарядке обычно выгоднее применять attack move.

Чтобы эффективно применять технику отмены анимации атаки умениям, надо настроить все умения чемпиона на "Quick Cast". Эта настройка находится на вкладке "HOTKEYS". Её демонстрирует иллюстрация 3-6.

{caption: "Иллюстрация 3-6. Настройка 'Quick Cast All'", width: "100%"}
![Настройка Quick Cast All](images/Micromanagement/quick-cast.png)

Откройте вкладку "HOTKEYS" в настройках и нажмите кнопку "Quick Cast All". После этого все умения чемпиона будут применяться как только вы отпускаете кнопку Q, W, E или R.

Сравним алгоритм применения умений с опцией "Quick Cast" и без неё. Рассмотрим умение W Ashe. Чтобы его применить без опции "Quick Cast", надо выполнить следующие шаги:

1. Нажать кнопку W.

2. Игра покажет область, которую накроет залп стрел.

3. Навести курсор мыши на желаемую область.

4. Нажать кнопку W второй раз, чтобы Ashe выпустила залп.

С опцией "Quick Cast" шаги станут такими:

1. Нажать кнопку W и не отпускать.

2. Игра покажет область, которую накроет залп стрел.

3. Навести курсор мыши на желаемую область.

4. Отпустить кнопку W, чтобы Ashe выпустила залп.

Опытные игроки сначала наводят курсор мыши на желаемую область, а потом просто нажимают кнопку W. Такой приём ускоряет использование умений. Но чтобы выполнить его эффективно, надо точно помнить область действия каждого умения чемпиона.

I> Умения с [**зарядкой**](https://leagueoflegends.fandom.com/ru/wiki/Подготовка) (charging), во время которой чемпион может двигаться, не надо настраивать на Quick Cast. Пока идёт зарядка, чемпион может сменить свою позицию с помощь заклинания призывателя [скачок](https://leagueoflegends.fandom.com/ru/wiki/Скачок_(заклинание_призывателя)) (flash). Использовать скачок неудобно, пока зажата клавиша умения с Quick Cast. Намного легче нажать кнопку умения один раз, затем использовать скачок и потом нажать кнопку умения второй раз. Примеры таких умений: Vi — Q, Viego — W.

Тренируйте технику отмены анимации атаки умением в "Practice tool". Постарайтесь хорошо запомнить область действия всех умений вашего основного чемпиона.
